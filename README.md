# Portfolio Site

Static multi-page portfolio rebuilt into a darker, motion-heavy presentation for GitHub Pages.

## Main pages

- `index.html` - animated landing page with interactive bookshelf and book-open transition
- `writing.html` - writing overview page for both books
- `dancing-on-jupiter.html` - dedicated page for Dancing on Jupiter
- `summer-of-06.html` - dedicated page for The Summer of 06
- `apps.html` - showcase for VERVE, CHERIFLIX, and BOOKLI
- `photography.html` - Tasmania photography page
- `prints.html` - print/store page linked to VanVakarnee
- `contact.html` - contact and enquiry page

## Shared assets

- `assets/css/styles.css` - dark theme, bookshelf visuals, cursor effects, responsive layout
- `assets/js/site.js` - GSAP reveals, custom cursor, particle background, tilt cards, page transitions, book-open animation

## Upload to GitHub Pages

1. Put all files in the root of your repository.
2. Commit and push.
3. In GitHub, open `Settings > Pages`.
4. Deploy from the root of your main branch.

## Before publishing

- Replace the placeholder contact email in `contact.html`.
- Replace placeholder descriptive copy anywhere you want more exact project or book details.
- Keep external links current, especially `https://vanvakarnee.com`.
